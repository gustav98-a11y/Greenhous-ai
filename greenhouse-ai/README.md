# 🌿 Greenhouse AI

Din smarta AI-odlingsassistent för växthus och köksträdgård.

## Funktioner
- AI-kameraanalys av växter (Claude Vision)
- Tillväxtlogg och skördespårning
- Sensorer & IoT-integration
- AI-chatbot för odlingshjälp
- Supabase-backend med inloggning

## Supabase är redan konfigurerat
URL och API-nyckel är inbyggda i appen.

## Lokal utveckling
```bash
npm install
npm run dev
```

## Deploy till Vercel
Koppla detta GitHub-repo till Vercel — byggs automatiskt vid varje push.
